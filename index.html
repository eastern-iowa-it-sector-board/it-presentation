Examples of I.T. in everything

Let's discuss solutions for hypothetical problems.

Then, we can identify how IT affects the solutions.

  • Each hypothetical is DOWN, cover as many as needed
  • Give LOTS of time for discussion
  • Each item mentions several solutions including those with less IT resources involved. Be sure to discuss pros and cons, and try to advance through the historical progression

Security

How can we prevent theft from a parking lot?

  • Add security guards and gates
  • Add proximity card access
  • Add security cameras and a central guard station
Be sure to mention how much more efficient a centralized solution can be.

Inefficiency

How can we manage a shared resource, like a library book?

  • With a librarian and physical check-out cards
  • With a digital check-out system
  • With online access to the inventory
Online inventory systems can even enable less driving through searches done before physically arriving. You can touch on how this might encourage less human interaction though, and this factor should be considered when creating new technology such as this.

Risk

How can you make a password more secure?

  • Add special characters
  • Add more length
  • Use two factor authentication technologies
Be sure to mention that just sustituting zeros for O's can easily be replicated by computers. Also that password length affect how hard it is to brute force attack the password. And finally how strength requirements might backfire and cause people to write them down more often.
